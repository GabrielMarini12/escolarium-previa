Conhecimento

Projeto

Este projeto foi idealizado pelo professor Carlos Paleo para as aulas de Programação Web do semestre 2022/1 do curso de Análise e Desenvolvimento de Sistemas na UNIFTEC.

Área de livros

Essa área permite que os usuários possam consultar e fazer donwload de livros de forma rápida, através de uma biblioteca virtual disponibilizada pela instiuição.

Área de caronas

Essa área permite que os usuários possam, de forma interativa, se disponibilizarem ou solicitarem carona.

Integrantes

O projeto foi desenvolvido pelos alunos da instituição da faculdade Uniftec de NH, compostos pelos integrantes: Thiago Heilmann, Gabriel Marini, Julia Vargas, Pedro Santana e Gustavo Canisio.
